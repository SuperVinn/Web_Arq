Alimentación

5V, 3.3V y GND organizados con énfasis en conexiones seguras.

monitoreo_pines.ino (Arduino IDE) 
// Configuración de arquitectura Harvard/RISC
const int ledPin = 13; // Pin digital integrado

void setup() {
  pinMode(ledPin, OUTPUT);    // Configura el pin 13 como salida digital
  Serial.begin(9600);         // Inicializa la comunicación serial a 9600 bps
}

void loop() {
  digitalWrite(ledPin, HIGH); // Envía 5V al pin (ENCENDIDO)
  Serial.println("LED: Encendido");
  delay(1000);                // Retardo de 1 segundo
  
  digitalWrite(ledPin, LOW);  // Envía 0V al pin (APAGADO)
  Serial.println("LED: Apagado");
  delay(1000);                // Retardo de 1 segundo
}

Programación

Arduino se programa habitualmente en C/C++ mediante el IDE de Arduino o entornos como PlatformIO. Algunas placas soportan MicroPython o CircuitPython; también es posible programarlas desde JavaScript usando Firmata y bibliotecas como Johnny-Five, o con Rust y toolchains específicos. El flujo típico consiste en escribir el sketch, compilar y subirlo vía USB al bootloader de la placa. Para empezar, el IDE oficial y los ejemplos preinstalados facilitan la curva de aprendizaje.
